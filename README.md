# Análise de Textos - Avaliação a Distância 1

## Descrição do texto utilizado
O texto utilizado é de um dataset da plataforma Hugging Face, composto pelo resumo de diversas notícias da CNN em português brasileiro [(celsowm/cnn_news_ptbr)](https://huggingface.co/datasets/celsowm/cnn_news_ptbr). Deste dataset foram selecionados 30 documentos para se fazer a análise.

## Etapas realizadas
- Geração do documentos.csv  
- Segmentacao e Limpeza  
- POS Tagging  
- NER  
- Análise dos Dados

Todos os notebooks utilizados estão na raiz do projeto.
